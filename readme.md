# BMW Sales Dashboard (SQL + Power BI)

## Project Overview

This project presents an interactive **BMW Sales Dashboard** developed using **SQL for data analysis** and **Power BI for visualization**.

The objective of this project is to analyze BMW car data to identify **price trends, model performance, revenue insights, and mileage comparisons** across different years.

SQL was used to perform data exploration and calculations, while Power BI was used to create an **interactive dashboard with dynamic filters and visual insights**.

---

## Tools & Technologies

* **SQL** – Data querying and analysis
* **Power BI** – Dashboard creation and visualization
* **DAX** – Measures and dynamic calculations
* **CSV Dataset**

---

## SQL Analysis

Using SQL, several queries were written to extract insights from the dataset, including:

* Total revenue generated from BMW car sales
* Average car price calculation
* Average mileage per model
* Maximum price to identify the most expensive BMW model
* Model-wise comparisons
* Trend analysis by year

These queries helped prepare the data before visualization.

---

## Power BI Dashboard Features

The Power BI dashboard includes:

* **Total Revenue KPI**
* **Average Car Price KPI**
* **Most Expensive BMW Model**
* **Average Mileage per Model (Bar Chart)**
* **Price vs Year Trend (Line Chart)**
* **Dynamic Year Slicer**
* **Dynamic Year Text Display**
* **Insight Section explaining the analysis**

---

## Key Insights

* BMW **3 Series provides the highest average mileage** among the models.
* Larger models such as **X series show comparatively lower fuel efficiency**.
* Car prices show **significant growth in recent years**.
* Certain BMW models contribute more to overall revenue.
